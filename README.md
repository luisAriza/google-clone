# Clon de Google

En este proyecto se _maquet√≥_ el buscador de Google con **HTML** y **CSS** aplicando _breakpoints_ para hacerlo _resposivo_ y poder verlo en mobile, tablet y desktop.

Para clonar el proyecto crea una carpeta llamada google_clone:
```sh
mkdir google_clone
```
Luego ingresa a ella:
```sh
cd google_clone
```
En la carpeta creada ejecuta lo siguiente:
```sh
git@github.com:luisAriza/google_clone.git
```
